Sistema de Gestión Escolar (Fullstack)

Este proyecto integra un diseño de base de datos relacional optimizado con una interfaz web funcional para la administración de registros educativos.

**Base de Datos y Normalización**
La base de datos fue diseñada en **MySQL** aplicando la **3ra Forma Normal (3NF)** para garantizar la integridad de la información y eliminar la redundancia.
* **Tablas principales:** Alumnos, Profesores, Materias, Grados, Salones y Calificaciones.
* **Integridad:** Uso de llaves primarias (PK) y foráneas (FK).

## 🌐 Aplicación Web (CRUD)
Desarrollé una interfaz funcional en **PHP** que interactúa en tiempo real con la base de datos, permitiendo realizar operaciones de:
* **Creación:** Registro de nuevos alumnos y materias.
* **Lectura:** Consulta dinámica de listas y promedios.
* **Actualización:** Edición de información académica.
* **Eliminación:** Depuración de registros.
